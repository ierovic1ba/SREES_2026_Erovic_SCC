Dijagrami (izvor za OpenDraw / LibreOffice Draw)
================================================

Fajlovi:
  - algorithm_flowchart.svg  : dijagram toka algoritma proracuna kratkog spoja
  - case9_oneline.svg        : jednopolna shema WSCC 9-sabirnicke mreze (case9)

Kako dobiti .odg (OpenDraw) fajl:
  1. Otvori LibreOffice Draw.
  2. File -> Open -> odaberi .svg fajl (Draw ga ucita kao editabilne oblike).
  3. Po potrebi doradi/rasporedi elemente.
  4. File -> Save As -> ODF Drawing (.odg)  -> snimi u ovaj folder.

Napomena: SVG je vektorski pa se moze slobodno uredjivati i skalirati u Draw-u.
